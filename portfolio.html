Project summaries (in reverse chronological order):

  • Project DHPS: Personal project I worked on over the summer between work hours. Made from programmer art, but contains a scratch-built dungeon generator, fully autonomous AI adventurers and RTS-like AI goons for the player to control. A few light building elements are also sprinkled in.

  • Blacksite Theta: Team Class Capstone project, when I wasn't working through my responsibilities as narrative lead on this project I lent my hand to building some of the backend systems, namely audio management and dialog implementation.

  • Lucidus: Team Game Jam project, for my addition to the project I made a melee-based arena combat scene and the animated cutscenes.

  • Root Access: Solo Game Jam project, with the added challenge of using only a single Csharp file. While the end result isn't exactly messy, it proves just why keeping code clean on larger projects is so important.

  • Checkers: Solo Personal project done under a self imposed time limit of 24 uninterrupted hours. Csharp ended up being utilized far more than the Unity engine for this challenge.

  • AkimBro: Solo Game Jam project where I experimented with using Character Controller instead of Rigidbody, along with a dynamic music system and some modified FPS controls.

  • Gorgon Stealth: Game Jam project where I helped a new club member realize his idea of a medusa sneaking around a maze trying to avoid turning the wandering humans inside to stone.

  • Operation Big Break: Solo Dev project that I coded and then re-coded over the course of about a year's worth of spare time. More or less aping Cave Story, but manages to recreate a vast majority of the mechanics. Notably, my codebase for this project contains 257KB worth of .cs files, my largest to date.

  • Agrav Racer: Solo developed class project; a top-down 2D arcade racer with drifting and driving backwards being core mechanics of beating the game.

  • Chain Game: Game Jam project partnered with Lucas Ackman, which was an experiment with recreating the Just Cause grappling hook tool.

  • Let's Make Some Noise: Solo Game Jam project, a simple top down arena combat game.

  • No More Assets: Solo Game Jam project where I fully embraced the concept of programmer art to mess around with state machines.

  • Requiem: The Lost Voices - I made minor contributions to this game jam project, first experience with 3D Unity.
